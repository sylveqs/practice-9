# 📚 Библиотечный портал

Веб-приложение для управления библиотечным фондом с полноценной системой аутентификации пользователей. Реализовано на FastAPI с использованием ванильного JavaScript.

## 📋 Функциональные возможности

### Пользовательский интерфейс
- **Главная страница** — стартовая точка входа с навигацией к формам регистрации и авторизации
- **Регистрация** — форма создания учетной записи с валидацией полей
- **Авторизация** — форма входа с проверкой подлинности учетных данных
- **Личный кабинет** — защищенная страница с данными пользователя

### Системные возможности
- Проверка уникальности имени пользователя и email при регистрации
- Валидация пароля (минимальная длина — 6 символов)
- Автоматическая авторизация после успешной регистрации
- Информативные сообщения об ошибках при некорректном вводе
- Управление сессией пользователя

### Документация
- Автоматически генерируемая документация Swagger UI по адресу `/docs`

## 🚀 Запуск проекта

### Требования
- Python 3.11+
- pip

### Установка и запуск

```bash
# Клонирование репозитория
git clone <repository-url>
cd tbm

# Backend
cd backend
python -m venv venv
# Активация виртуального окружения
# Windows:
venv\Scripts\activate
# Linux/Mac:
source venv/bin/activate

pip install -r requirements.txt
uvicorn main:app --reload --host 0.0.0.0 --port 8000

# Frontend (в новом терминале)
cd frontend
python -m http.server 3000
```

Приложение будет доступно по адресу: `http://localhost:3000`

## 🔌 API Endpoints

| Метод | Endpoint | Описание |
|--------|----------|----------|
| POST | `/api/register` | Регистрация нового пользователя |
| POST | `/api/login` | Вход в систему |
| GET | `/api/user/{user_id}` | Получение данных пользователя |

### Пример запроса
```json
POST /api/register
{
    "username": "reader123",
    "email": "reader@library.com",
    "password": "secret123"
}
```

## 🛡️ Безопасность

- Хеширование паролей с использованием SHA-256 и соли
- Валидация данных на стороне клиента и сервера
- Уникальность username и email
- Настроенный CORS
- Централизованная обработка ошибок

## 🎨 Дизайн

- Теплая цветовая гамма в коричневых тонах
- Шрифты Merriweather и Lora (Google Fonts)
- Плавные анимации и hover-эффекты
- Адаптивная верстка
- Стилизация под классическую библиотеку

## 📦 База данных

SQLite с таблицей `users`:
- `id` (INTEGER PRIMARY KEY)
- `username` (TEXT UNIQUE)
- `email` (TEXT UNIQUE)
- `password_hash` (TEXT)
- `created_at` (TIMESTAMP)
